# Evidence and Confidence Model

## Confirmed observation

동일 조건에서 직접 관찰하거나 로그·API 결과로 재현된 사실입니다.

## Documented behavior

Microsoft 공식 문서 또는 제품 문서에 명시된 동작입니다.

## Technical inference

수집된 증적과 시스템 동작을 기반으로 한 기술적 추론입니다.

## Open hypothesis

추가 재현 또는 검증이 필요한 가설입니다.

## Excluded cause

동일 조건 비교 또는 반증 테스트를 통해 1차 원인에서 제외한 항목입니다.
